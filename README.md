## Esse é o Brain Dump, um site onde vc escreve o que quiser
baseado no Site [akitaonrails.com](https://akitaonrails.com/)do Fábio Akita,  o Brain Dump tem o mesmo intuito, você escreve algo pra ler depois, escreva do jeito que quiser.
a única diferença é que no 'Akita On Rails', ele posta e todos que entrarem no site conseguem ter acesso, mas o brain dump é só você quem vê 
é isso,
